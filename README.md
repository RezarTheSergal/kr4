# Простой чат-бот с помощью React

Простой чат-бот (без сервера) — ввод текста и отображение ответа-заглушки.

## Функционал

- Ввод текста через поле ввода.
- Отправление сообщения по нажатию кнопки или клавиши Enter.
- История сообщений
